数当てゲーム

このゲームは、コンピュータが選んだ1から10までの数字をピタリと当てるシンプルなゲームです。テキストボックスに数字を入力して「回答」ボタンを押すと判定が出ます。間違えても「もっと大きい」「もっと小さい」といったヒントが出るので、それを頼りに推理できます。回答できるチャンスは全部で3回までです。ヒントを上手く活用して、見事3回以内にクリアできるか、ぜひチャレンジしてみましょう。

数当てゲームのページ：kazuate.html

各授業回の学習事項

第1回 WWW と HTML

今回はHTMLの基本を学んだ。そもそも簡単な方な言語であり、かつ高校時代に一度やったことがある言語なので理解しやすかった。忘れていたところも多くあったが、思い出すことができたと思う。また、ミスがあってもある程度は表示されてしまうということを初めて知った。気をつけようと思う。

演習の成果：演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回はCSSの基本を学んだ。HTMLと一緒に編集をしなければならず、決まり事も多かったので少し難しかった。また、数値や色などを細かく設定できるのでなかなか時間がかかると思った。しかし、自分で好きにWebページをデザインしていくということが楽しいと感じた。

演習の成果：演習2-3: ウェブブラウザ説明ページの編集

第3回 JavaScript の基本

今回はJavaScriptの基本を学んだ。web開発ツールをうまく使うことでスムーズにコードを作り上げていくことができるとわかった。JavaScriptはHTMLやCSSと違って、プログラミング言語なのでまだ理解ができていないところがいくつかある。次回の作業から本格的に難しくなると思うので学んだことをしっかり復習しておきたい。

演習の成果：演習3-6 :オブジェクトの参照

第4回 DOM と JavaScript

今回は、イベント処理の前段階として、DOMとJavaScriptの関係を学んだ。前回までと比べて難易度が上がったと感じた。宿題のJavaScript プログラムを編集して，検索ボタンを押したらユーザが入力した検索キーをコンソールに表示する処理というのができなかった。理解が浅く自力でできないことがまだ多いので次回のイベント処理のためにしっかり復習する必要があると感じた。

演習の成果：演習4-10 :足し算ページ
